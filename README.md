Tethys:analyze_eeg dperi$ analyze_muse_monitor_data.py -h

usage: analyze_muse_monitor_data.py [-h] [-c CSV_FILE] [-v VERBOSE] [-d]

  -h, --help            show this help message and exit
  -c CSV_FILE, --csv_file CSV_FILE
                        CSV file to read)
